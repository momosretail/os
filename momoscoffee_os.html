구글시트 연동

Apps Script 웹 앱 URL을 입력하면 시트의 메뉴를 읽어오고, 이 화면에서 추가·수정한 내용이 시트에 자동 기록됩니다.

  1. 구글시트 → 확장 프로그램 → Apps Script 에 연동 스크립트 붙여넣기
  2. setupSheet 실행 후 → 배포 → 새 배포 → 웹 앱
  3. 액세스 권한 '모든 사용자' 로 배포 → 웹 앱 URL 복사

메뉴 추가

입력한 내용은 [저장] 시 구글시트에 기록됩니다. 한 칸에 여러 항목은 줄바꿈으로, 구조 항목은 | 로 구분하세요.
